Useful Websites

Some websites I use for making stuff!

  • Ezgif - A website for creating GIFs out of images and videos.
  • 98.css/XP.css - A CSS library aiming to faithfull recreate Windows 98/XP.
  • Window Creator - A website for creating Windows styled error popups.
  • blinkies.cafe - A website for customizing blinkies.
  • Button Makes - A website for making 88x31 buttons.
  • Cobalt - A website for downloading media from websites like Twitter, Youtube, etc.
